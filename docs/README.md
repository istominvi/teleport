# Teleport — интернет без границ

Teleport — это сервис, который автоматически выдаёт безопасные конфигурации доступа к зарубежным сайтам через Telegram-бота. Сервис полностью автоматизирован и не требует ручной настройки. Работает на базе Xray (3x-ui), Node.js и PostgreSQL.

## Функциональность

- Telegram-бот с простым управлением
- Регистрация пользователей и учёт активности
- Автоматическая выдача конфигураций
- Интеграция с Robokassa
- Система подписок (месяц, год)
- Реферальная система с бонусами
- Мониторинг серверов и управление нодами
- Админ-панель (в разработке)
- Рассылки, аналитика и трекинг трафика

## Технологии

- Node.js (Telegram Bot, REST API)
- PostgreSQL (структурированное хранилище)
- Xray + 3x-ui (серверы доступа)
- GitHub + Markdown-документация
